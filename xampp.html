How to install Xampp

Go to https://www.apachefriends.org/index.html

Download Xampp for whatever operating system you are using.

Install Xampp with default settings.

Change Apache config by opening httpd.conf

Find document root and change to your website directory.(careful about forward and back slashes)

save the file and restart apache if needed.

Go to localhost in web browser

How to setup MySQL

Click Start in XAMPP control panel for MySQL

Click Admin button to open up phpmyadmin

Select user Accounts

Select Add User Account

Put in a username, select localhost, put in a password

Select Global Privileges - Select all

click Go in the bootom right corner to confirm the account

How to get a domain name and Setup DNS

Go to https://www.freenom.com

Find domain name you like. Click check and select use own DNS

Go to https://www.namecheap.com and sign up for an account

Use the FreeDNS service and register your domain with their DNS servers

Use the DNS servers provided from NameCheap in the Freenom domain name purchasing

Purchase the domain and fill out all the information.

Go to your dashboard in NameCheap and select manage on your domain

Create an A record by searching google for what's my ip address

Copy your IP address into the record and then down below select dynamic DNS

download the software for Dynamic DNS

Open Zip File and Run the exe file

Create a new profile by using @ as the host, your domain name and the password generated in the advanced DNS section

Select update every 15 minutes and auto detect IP Address and save.

How to setup your router to forward port 80 to your computer.

1.Open Command prompt and type in ipconfig
  default gateway (your router)
  IPv4 Address (your computer)
2. Go to http://(default gateway IP)
3. Log In (your router login information may be on a sticker on the 
  top/back/bottom of your router)
4. Look for 'Port Forwarding' or 'Port Triggering'
5. 'Add' or 'Create'
6. If you are required to input a range, enter 80 - 80
7. Enter the IPv4 Address from step #1
8. Choose TCP if asked
9. Open XAMPP on your computer and start Apache if you haven't already,
  accept the message from your firewall.
